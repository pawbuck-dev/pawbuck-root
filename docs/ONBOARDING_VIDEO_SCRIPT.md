## Production Notes

### Visual Style
- **Color Scheme:** Use Pawbuck's teal (#0D9488 / #5FC4C0) as primary accent
- **Transitions:** Smooth fades between scenes (0.3-0.5 seconds)
- **Text Overlays:** 
  - Font: Outfit (matching app font)
  - Size: Large, readable
  - Position: Bottom third of screen
  - Background: Semi-transparent dark overlay for readability

### Audio
- **Background Music:** 
  - Upbeat, friendly, non-intrusive
  - Volume: 30% of narration
  - Royalty-free options: YouTube Audio Library, Epidemic Sound
- **Voiceover:**
  - Warm, friendly, professional tone
  - Clear pronunciation
  - Pace: ~150 words per minute

### Recording Tips
1. **Use Real Device:** Record on actual iPhone/Android, not just simulator
2. **Clean UI:** Use test data that looks professional
3. **Smooth Scrolling:** Practice transitions before recording
4. **Light Mode:** Use light theme for better visibility in video
5. **No Personal Data:** Use placeholder names, emails, pet names

### Editing Checklist
- [ ] Add fade-in/fade-out at start and end
- [ ] Add smooth transitions between scenes
- [ ] Sync text overlays with narration
- [ ] Add subtle zoom/pan effects on key screens
- [ ] Add background music (fade in/out)
- [ ] Add captions/subtitles
- [ ] Add App Store & Google Play badges at end
- [ ] Export in 1080p (1920x1080) MP4 format
- [ ] Keep file size under 50MB for easy sharing
