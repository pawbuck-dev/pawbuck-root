## Typography Reference

### Font Sizes
- **Large Heading:** 36-48px (Outfit Bold)
- **Medium Heading:** 24-32px (Outfit SemiBold)
- **Body Text:** 16-18px (Outfit Regular)
- **Small Text:** 12-14px (Outfit Regular)

### Text Overlay Styles
- **Background:** Semi-transparent dark overlay (rgba(0, 0, 0, 0.7))
- **Padding:** 16px horizontal, 12px vertical
- **Border Radius:** 12px
- **Position:** Bottom third of screen
- **Animation:** Fade in/out (0.3s)

## Animation Timing Reference

### Transitions
- **Screen Transitions:** 0.3-0.5s slide/fade
- **Button Press:** 0.1s scale (0.95x)
- **Modal Open:** 0.4s slide up from bottom
- **Notification:** 0.3s slide down from top

### Effects
- **Highlight Glow:** 0.5s pulse
- **Progress Animation:** 1s fill from 0% to target
- **Badge Appear:** 0.3s bounce scale
- **Text Fade:** 0.3s opacity

## Recording Checklist

### Pre-Recording Setup
- [ ] Use light theme for all screens
- [ ] Prepare test data (pets, care team, messages)
- [ ] Clear notification badges (or set specific counts)
- [ ] Set device to Do Not Disturb
- [ ] Use high-quality device (iPhone 14 Pro or similar)
- [ ] Ensure good lighting if showing device
- [ ] Close unnecessary apps

### During Recording
- [ ] Record in 1080p (1920x1080)
- [ ] Use 30fps (or 60fps for smooth animations)
- [ ] Record each scene separately
- [ ] Hold each screen for 2-3 seconds
- [ ] Make smooth, deliberate taps
- [ ] Avoid rapid scrolling
- [ ] Show loading states if needed

### Post-Recording
- [ ] Review all footage
- [ ] Trim unnecessary pauses
- [ ] Ensure smooth transitions
- [ ] Check text readability
- [ ] Verify all features are visible
- [ ] Export in MP4 format (H.264)
